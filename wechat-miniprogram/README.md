Official WeChat Mini Program (Beta) — MOSE
Development Period: During the 2026 Labor Day holiday
Core Features: Home page, reader, submission system with history, and message board — all fully synced with the official website.
Status: Ultimately not listed on the WeChat Mini Program platform due to strict review policies imposed on individual-subject accounts. The version that did go live was a simplified one, with all interactive features redirected to the official website.
Tech Stack: WeChat Native + Cloudflare Worker + mose.cc.cd domain proxy + Supabase REST API
